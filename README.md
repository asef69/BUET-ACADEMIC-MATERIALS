## 🚀 Major Projects

### Football Game - iGraphics Game Project (CSE 102)

* **Repository**: [iGraphics-Asef](https://github.com/asef69/iGraphics-Asef)
* **Technologies**: C++, iGraphics Library, Graphics Programming, Audio Integration
* **Course**: CSE 102 (Level 1 Term 1)
* **Description**: A 2D football penalty-saving game developed using the iGraphics library, featuring interactive gameplay, score tracking, and graphical user interfaces.

#### Features

* Interactive 2D football gameplay
* Keyboard and mouse-based controls
* Multiple screens (Home, Instructions, About, Scoreboard)
* High-score tracking and persistence
* Sound effects and background music integration
* Custom graphical assets and animations
* Event-driven game state management

#### Impact

* Demonstrates fundamental game development concepts
* Showcases graphics programming and event handling
* Implements file-based data persistence
* Highlights problem-solving and software design skills using C++

### Digital Logic Simulator - Java Desktop Application

* **Repository**: [Digital-Logic-Simulator](https://github.com/asef69/Digital-Logic-Simulator)
* **Technologies**: Java, Maven, CSS, Object-Oriented Programming
* **Course**: Academic Software Engineering Project
* **Description**: A digital circuit simulation platform that allows users to design and test logic circuits using common digital logic gates and components.

#### Features

* Simulation of digital logic circuits
* Support for:

  * AND Gate
  * OR Gate
  * NOT Gate
  * NAND Gate
  * NOR Gate
  * XOR Gate
  * XNOR Gate
* Signal and connector management
* User authentication system
* Dashboard and project workspace
* Modular architecture for extensibility
* Maven-based project management

#### Impact

* Demonstrates strong object-oriented design principles
* Models real-world digital logic systems
* Showcases software architecture and modular development
* Strengthens understanding of computer hardware fundamentals

### Core BUETians - Your BUETian Social Media App (CSE 216 Project)

* **Repository**: [Core-BUETians](https://github.com/asef69/Core-BUETian--Your-BUETian-Social-Media-App)
* **Technologies**:

  * Django
  * Django REST Framework
  * Django Channels
  * PostgreSQL
  * Redis
  * React
  * Vite
  * Axios
  * JWT Authentication
  * WebSockets
* **Course**: CSE 216 Database and Full-Stack Project
* **Description**: A full-stack social media platform designed exclusively for BUET students, combining social networking, communication, collaboration, and community engagement in a single ecosystem.

#### Features

* User registration and authentication
* JWT-based secure login system
* User profiles and account management
* Social feed with posts and interactions
* Like and comment functionality
* Hashtag-based content organization
* Real-time chat using WebSockets
* Group creation and management
* Discussion forums
* Marketplace for buying and selling
* Notification system
* Advanced search functionality
* RESTful API architecture
* API documentation with Swagger and ReDoc
* PostgreSQL database integration
* Redis-powered real-time communication

#### Impact

* Demonstrates full-stack software engineering capabilities
* Implements scalable client-server architecture
* Utilizes real-time communication technologies
* Integrates modern frontend and backend frameworks
* Showcases database design and API development expertise
* Represents a production-style web application with multiple interconnected modules
